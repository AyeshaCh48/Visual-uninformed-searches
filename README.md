# 🧠 AI Pathfinding Visualizer

This project is a **Python-based GUI application** that demonstrates different **Artificial Intelligence search algorithms** using a grid-based environment. It visually shows how each algorithm explores nodes and finds the path from a start node to a target node.

Built using **Tkinter**, this tool helps in understanding and comparing classic search algorithms in an interactive way.

---

## 🚀 Features

- Visual representation of search algorithms
- Grid-based environment with obstacles
- Animated exploration of nodes
- Path tracing from source to destination
- Multiple AI search algorithms implemented

---

## 🧩 Algorithms Implemented

1. **Breadth-First Search (BFS)**
2. **Depth-First Search (DFS)**
3. **Uniform Cost Search (UCS)**
4. **Depth-Limited Search (DLS)**
5. **Iterative Deepening DFS (IDDFS)**
6. **Bidirectional Search**

---

## 🎮 How It Works

- A grid (12x12) is created
- Start node (S) is marked in **green**
- Target node (T) is marked in **blue**
- Obstacles are shown in **black**
- Algorithms explore nodes:
  - **Yellow** → Currently visiting
  - **Cyan** → Discovered nodes
  - **Red** → Final path
